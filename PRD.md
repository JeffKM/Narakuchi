# 나라카찌 (Narakatchi) — PRD

> 나라카 컨셉카페 팬게임. 도트 그래픽의 가벼운 데일리 교감·수집 게임.
> 도메인 용어는 [CONTEXT.md](./CONTEXT.md), 되돌리기 어려운 결정은 [docs/adr/](./docs/adr/) 참조.

## 1. 개요

- **제목**: 나라카찌 (나라카 + 다마고찌)
- **장르**: 데일리 교감(다마고찌 톤) + 체키(포토카드) 수집
- **메인 캐릭터**: **옥자**(데모) → 확장: 바나·멜·미호. **시온이**는 옥자 곁의 교감 가능한 펫(서브).
- **엔진**: Godot 4
- **플랫폼**: 웹(HTML5 export) · 모바일 세로(portrait) · PWA 설치 지원
- **저장**: 로컬(localStorage)만, 서버/계정 없음 (데모 범위)
- **기간**: 약 1주 (옥자님 대상 가능성 데모)

## 2. 목적과 승리 조건

옥자님이 펀딩으로 만들고 싶어하는 본격 **미연시**는 착수 시점이 불확실하다. 그 전에, 빠르게 만들어 **가능성을 증명**하고 카페 **마케팅/방문자 참여**에 즉시 도움이 되는 가벼운 게임을 만든다.

**1주 데모의 최우선 승리 조건 = "감정 임팩트 + 공유성".**
- 옥자님이 화면을 본 순간 *"내가… 살아 움직인다"*는 감정적 반응을 끌어낸다 → 신뢰 확보 + 미연시 그린라이트.
- 그 자체로 링크/QR로 가볍게 배포 가능한 마케팅 장난감이 된다.
- *1주에 마케팅 ROI 증명은 불가능하므로, "깊은 시스템"이 아니라 "감정 + 공유"에 집중한다.*

## 3. 핵심 컨셉: 체키가 심장이다

나라카의 실물 굿즈 **체키**(함께 찍은 사진을 꾸민 포토카드)를 디지털 수집물로 옮긴다. 체키는 한 시스템에 5가지 역할을 겸한다: **보상 · 수집 · 공유 · 마케팅 전환 · 호감도(미연시 브리지)**.

- **공유성**: 컬렉션을 자랑 → SNS 공유 → 무료 바이럴.
- **데일리 루프**: 매일 접속 = 오늘의 체키 한 장 (앱테크).
- **마케팅 전환**: 디지털 체키 → 실물 체키 욕구 → 카페 방문.
- **콘텐츠 확보**: 나라카는 이미 **1년치 이벤트 데이 의상 아카이브**를 보유. 사진을 카드로 옮기면 콘텐츠가 마르지 않는다.

## 4. 게임플레이 — 데일리 루프 (옥자·시온이 교감)

"키운다"가 아니라 **매일 나라카에 들러 옥자(와 시온이)와 함께 시간을 보내며 친해지는** 톤. 다마고찌의 가벼운 데일리 구조는 유지.

```
[매일 접속] 나라카 입장 → 옥자가 맞이함(출석 보상)
  ── 옥자 교감 (하단 4버튼, 모두 호감도↑ · 획득 아님) ──
  → 🃏 체키 주문 : 옥자에게 체키를 조름 (호감도↑)
  → 🍷 음료 주문 : 시그니처 음료 제조 연출 (기분 회복 + 호감도↑, 선호 보너스)
  → 💬 대화  /  🎁 선물 : 선택지 → 호감도 분기 (긍정 위주)
  → (옥자 직접 터치 → 실시간 표정 반응 + 소폭 호감도)
  ── 시온이 교감 (시온이 탭 → 하단 4버튼이 시온이용으로 전환) ──
  → 🃏 체키 주문 · 🍖 간식 · 🧶 놀기 · 🤚 쓰담 → 시온이 호감도↑
  ── 보상 (오더로 바로 안 줌) ──
  → 호감도 게이지 가득 (스태미나 게이트, 1~3분) → "오늘의 체키" 자동 획득
        (미보유 의상 우선 = 일반 / 중복 → 나비 승급)
  → 컬렉션북 전시 + 한 장 이미지로 공유
```

**핵심**: 체키는 *오더 버튼*으로 바로 주지 않는다. 4버튼은 **호감도 게이지를 채우는 교감**일 뿐이고, **게이지가 가득 찼을 때만 체키 1장이 자동 획득**된다. 옥자와 시온이는 **각자 호감도·게이지·체키**를 가진다(같은 시스템 복제).

### 4.1 미터 (딱 2개 + 기분)
- **호감도 게이지**: 교감으로 차는 진행도. 가득 → 체키 1장 자동 획득. (옥자/시온이 각각)
- **스태미나**: 하루 행동량 게이트. 소진 시 그날 종료(매일 회복). → 가벼움 + 재방문.
- **기분**: `행복 → 보통 → 시무룩`. 방치 시 시무룩(표정 처짐, 호감도 획득 소폭↓). **아프거나 떠나지 않음**. 들르면 즉시 회복.

### 4.2 교감 액션
**옥자 모드 (하단 4버튼, 전부 호감도↑)**
| 버튼 | 내용 |
|---|---|
| 🃏 체키 주문 | 옥자에게 체키를 조름 → 포즈/반응(호감도↑). *획득 아님* |
| 🍷 음료 주문 | 시그니처 음료 제조 연출 → 기분 회복 + 호감도↑, 선호 보너스 |
| 💬 대화 | 짧은 대사 + 선택지 → 호감도 분기(좋은 선택=특별 반응) |
| 🎁 선물 | 선물 선택 → 호감도 분기 |
| (옥자 터치) | 실시간 표정 반응(부끄/웃음) + 소폭 호감도 |

**시온이 모드 (시온이 탭 → 버튼 전환)**: 🃏 체키 주문 · 🍖 간식 · 🧶 놀기 · 🤚 쓰담 → 시온이 호감도↑.

### 4.3 대사·관계·선물 — 다마고찌 톤 (짧게)
**짧은 대사 원칙**: *표정이 9할, 대사는 한 줄.* 상시 대화창 없음.
- **한 줄 티커**: 화면 하단 한 줄에 옥자/시온이 보이스 반응이 뜸(상황 반응). 예) `옥자: 흥, 또 왔어.` 화면 공간 추가 0 — 다마고찌의 상태 티커를 캐릭터 보이스로.
- **대화 팝업**: `대화` 버튼 탭 시에만 순간 오버레이(한 줄 질문 + 두세 단어 선택지 + 한 줄/이모트 반응) → 닫힘. 선택은 **긍정 위주 분기**(나쁜 선택도 *덜 좋은 반응*까지, 벌점 X). 미연시 맛은 *양념*으로만.

**관계 단계 (존댓말 → 반말, 미연시 핵심)**
- 호감도가 쌓이면 관계 단계 상승: `손님(존댓말 + "{닉}님") → 단골(반말 + 닉네임/애칭) → 마음 연 사이` (데모 2~3단계).
- 첫 접속 시 **닉네임 입력** → 옥자가 그 이름을 불러줌(몰입 폭발 + 데모 임팩트).
- 단계 상승 시 **특별 컷인 한 순간** — *옥자가 존댓말을 푸는 그 장면*이 데모 감정 절정. (예: "{닉}님… 아니, {닉}. 😳 이제 반말할게. ……착각 마.")

**선물 선호**: 작은 선물 세트(3~4종), 캐릭터별 좋아함/보통. 맞으면 호감 ↑↑ + 특별 표정, 안 맞아도 손해 X. **시온이 간식 = 옥자 최고 호감**(옥자가 시온이를 사랑 → 두 캐릭터를 엮는 숨은 정답).

### 4.4 벌 없는 설계 (브랜드 안전 원칙)
방치 시 "시무룩"한 정도의 가벼운 압박 + 재방문 시 "기다렸어"의 긍정 정서만. 연속출석이 끊겨도 페널티 없이 보너스만 사라진다.

### 4.5 수치 밸런스 (v1 시작값 — 플레이테스트로 조정)
설계 의도: **세션 1~3분 · 체키는 희소(약 1주에 1장) · 첫 체키는 즉시 · 반말 전환은 장기 보상**.

**스태미나 (세션 길이)**
- 최대 **30**, 매일 접속 시 풀 충전. 액션당 **5** → **하루 6액션**(터치 무료). 옥자/시온이 **스태미나 공유**.

**호감도 획득 (액션별)**
| 액션 | 호감도 |
|---|---|
| 🍷 음료 주문 | +12 (선호 음료 +18) |
| 🃏 체키 주문 | +10 |
| 💬 대화 | 좋은 선택 +15 / 평범 +8 |
| 🎁 선물 | 맞음 +20 / 보통 +10 · 시온이 간식→옥자 +25 |
| 👆 터치(무료) | +2, 세션당 +10까지 |
| 🐱 시온이(간식/놀기/쓰담) | 각 +10 (선호 간식 +15) |

**게이지 → 체키**
- 옥자 게이지 **300** / 시온이 게이지 **200**. 가득 = 체키 1장. 집중 시 ~50/일 → **약 1주에 1장**.
- ✨ **첫 방문 기념**: 닉네임 입력 직후 옥자가 **지뢰계 일반체키 1장 즉시 증정**(빈 컬렉션 방지 + 히어로 즉시 노출).

**관계 단계 (누적 옥자 호감도)**
| 단계 | 임계 | 변화 |
|---|---|---|
| 손님 | 0 | 존댓말 + "{닉}님" |
| 단골 | **600** (~2주) | 반말 + 닉네임 |
| 마음 연 사이 | **2000** (~6주) | 속내·애칭·특별 대화 |
> ⚠️ 데모 시연 세이브는 *누적 ~560*으로 미리 채워 첫 세션에 **반말 전환 컷인**이 터지게 한다.

**나비 / 출석 / 코인 / 기분**
- **나비 승급**: 같은 의상 중복 1 = 나비 조각 +1. **조각 3 + 해당 일반 보유 → 나비 승급** (+ 7일 출석 마일스톤 조각).
- **연속출석**: 매일 +코인, 3일/7일 마일스톤 보너스. 끊겨도 벌점 0, 보너스만 리셋.
- **코인**: 액션·출석으로 획득 → 선물 구매(10~20코인, 하루 1~2개).
- **기분**: 24h+ 미접속/포만감↓ → 시무룩(호감도 획득 −20%), 케어 1~2회로 회복.

## 5. 캐릭터 (→ ADR 0002)

- **옥자 (메인 교감·수집 캐릭터)**: 화면의 라이브 캐릭터. 기본 의상 1벌, 이벤트 의상은 체키(정적)로 수집. 교감 4버튼.
- **시온이 (교감 가능한 펫 + 수집 캐릭터)**: 옥자 곁의 펫. 탭하면 시온이 교감 모드로 전환, 자체 호감도·체키(의상 2종).
- **확장**: 바나·멜·미호(메인) + 코코·선아·수아·규종이(펫). 데모엔 잠금.
- **체키 수집 모델**: `캐릭터 × 이벤트 데이` = 한 칸. 이벤트끼리 등급을 매기지 않음(동등한 세트). **캐릭터마다 칸 수는 다를 수 있다**(데모: 옥자 5칸, 시온이 2칸).
- **등급(오직 프레임으로만 구분, 캐릭터 아트 동일)**:
  - **일반체키** = 표준 프레임(공용). 평소 플레이로 자주.
  - **나비체키** = 그 이벤트 테마 프레임(예: 항공 데이 → 공항/비행기). 마일스톤·연속출석의 정성 보상(레어).
- **중복 → 나비 승급**: 같은 의상 일반체키가 또 나오면 나비 승급 재료가 된다(가챠 꽝/중복 스트레스 0).
- **현금 없음**: 게임 내부는 무과금. 현금은 "현실 방문/굿즈 → 게임 한정 체키 해금"의 오프라인 보상 다리로만(데모엔 컨셉만).

## 6. 화면 / UX (게임기 셸 + 270×480 세로)

- **게임기 셸 (정체성·공유성·감정의 핵심)**: 화면을 **핑크 나라카 게임기 프레임**으로 감싼다. **게임 화면은 270×480 그대로 유지**(쪼그라들지 않음), 요즘 폰의 9:16보다 긴 **세로 여백을 "버튼 덱"**으로 채움.
  - **물리 버튼 기능 연결 + 화면 터치 하이브리드** (데드 탭 0): `Ⓐ` 확인/오더 · `Ⓑ` 취소/뒤로 · `D패드` 메뉴·선택지 이동 · `Ⓝ` 홈/탭.
- **온보딩(첫 접속)**: 닉네임 입력 → 옥자가 맞이(존댓말) → **첫 방문 기념으로 지뢰계 일반체키 1장 증정**.
- **교감 화면(메인)**: 상단 HUD(기분/스태미나/호감도/코인) · 중앙 바 카운터 + 옥자(표정 스왑)·시온이 · 하단 **4버튼 [체키 주문][음료 주문][대화][선물]** · **맨 아래 한 줄 티커**(옥자/시온이 보이스).
  - 옥자 탭 → 터치 반응. **시온이 탭 → 하단 4버튼이 시온이 모드로 전환**(Ⓑ/옥자 탭으로 복귀).
  - `대화` → 순간 팝업(짧은 2지선다). 호감도 게이지 가득 → "오늘의 체키" 자동 획득 팝업.
- **컬렉션북 탭**: 캐릭터 선택(옥자 / 시온이 / 🔒잠긴 탭) → 이벤트 칸 그리드(일반/나비/빈칸/🔒) → 카드 탭 시 풀카드 확대 + [공유].
- **공유**: 카드 확대 화면 내에서 이미지 미리보기(워터마크 `@나라카` + QR 자리) → 저장/공유.
- **출석 팝업**: 접속 시 옥자가 맞이하는 연출 + 연속출석 보상.

## 7. 아트 규격 (요약 → ADR 0001)

- 베이스 캔버스 **270×480**, 정수배 스케일, Nearest 필터. 나라카 브랜드 앵커 **마스터 팔레트(~32색)** 강제 인덱싱.
- **옥자 라이브 = 표정 스왑 스탠딩 방식(미연시식, 쉬움)**: 표정/포즈별 **5~6장**(`기본/웃음/부끄/시무룩/제조중/말하기`)을 상황에 맞게 통째로 교체. 리깅 없이, **코드 트윈으로 살짝 둥실 흔들기·말할 때 톡톡·페이드**만 줘서 생동감 확보. 반신 영역 약 140×180.
- **시온이**: 48×48, idle + 교감 반응(간식/놀기/쓰담) 몇 종. 고양이라 그리기 쉬움.
- **체키 카드(정적)**: 120×180, 내부 아트 104×140 고정. 옥자 이벤트 의상 5 + 시온이 2.
- **파이프라인**: 실물 사진 → Midjourney/Gemini 도트화(표정만 바꿔 변형) → 목표 px nearest 축소 → 팔레트 인덱싱 → 수동 정리.
- 아이콘: 음료/선물/UI 24×24, 재화 16×16. 폰트: 갈무리11/9 (한글 도트, 라이선스 확인).

## 8. 마케팅 훅

- **공유 카드**: 모든 체키/컬렉션은 워터마크(@나라카 + QR)와 함께 이미지로 내보내기 → SNS 바이럴.
- **시그니처 메뉴 노출**: 음료 주문이 실제 메뉴(버건디 윗치 등) 홍보를 게임에 녹임.
- **QR/링크 배포**: 카페 내 QR 한 장, 인스타 링크 → 설치 마찰 0.
- **오프라인 다리(데모엔 컨셉)**: 잠긴 "한정" 슬롯 → "실제 방문/굿즈로 해금" 슬라이드로 확장성 제시.

## 9. 스코프 — IN / OUT

### ✅ 데모 MVP (반드시)
1. 교감 메인 화면: 옥자(표정 스왑 스탠딩) + 시온이 + 바 카운터 배경.
2. **옥자 4버튼**(체키 주문/음료 주문/대화/선물) + **옥자 터치 반응** + 음료 제조 연출 + 대화·선물 선택지 분기.
3. **시온이 교감 모드 전환**(체키 주문/간식/놀기/쓰담).
4. 미터: 호감도 게이지(옥자·시온이 각각) / 스태미나 / 기분(시무룩, 벌 없음).
5. **체키 획득 = 게이지 풀 시 자동** + 중복→나비 승급 + 연속출석→나비 프레임 해금.
6. 컬렉션북(옥자·시온이 × 이벤트 칸, 보유/빈칸/잠금) + 카드 확대.
7. 공유(이미지 내보내기 + 워터마크/QR 자리).
8. 출석/연속출석 + PWA(홈화면 추가, 세로 고정) + 로컬 저장.
9. 콘텐츠: **이벤트 5개** — 옥자 의상 **5세트** + 시온이 의상 **2세트** + 테마 프레임 5 + 표준 프레임 1. 옥자 표정 5~6장, 음료 3~4종, 선물 3~4종, 대화 분기 몇 토막.

## 10. 데모 진행 시나리오 (옥자님 앞에서)

1. 폰에서 링크 열기(설치 0) → 첫 화면에 **옥자가 카운터 너머에서 살아 움직이며 맞이함** = 첫 감정 타격("내가…").
2. 음료 주문 → **옥자가 시그니처 음료 제조 연출** = "버건디 윗치 만드는 거 봐".
3. 대화+선물 → **선택지에 따라 옥자 반응이 바뀜** = "이거 미연시 느낌이네".
4. 시온이 탭 → **버튼이 시온이용으로 바뀌고 시온이와 교감** = "우리 시온이도 있어!".
5. 교감 쌓여 게이지 풀 → **"오늘의 체키" 자동 획득**(실물 이벤트 의상 도트) = "이거 그날 옷이잖아".
6. 컬렉션북 → 모은 체키 + 잠긴 칸 → "이렇게 확장됩니다". 공유 버튼 → QR 카드 → "손님이 인스타에 올리면 홍보돼요".
