# 도트 아트 규격: 270×480 캔버스 + 마스터 팔레트 + 사진→도트 파이프라인

## 결정

게임은 **베이스 캔버스 270×480 px(9:16 세로), 정수배 스케일, Nearest 필터**로 렌더링한다. 모든 에셋은 나라카 브랜드 앵커(먹빛 블랙·버건디·블러드레드·앤틱 골드·캔들 옐로 등) 기반 **약 32색 마스터 팔레트로 인덱싱**해 통일한다. 아트는 직접 픽셀을 찍지 않고 **실물 이벤트 사진 → Midjourney/Gemini 도트화 → 목표 해상도로 nearest-neighbor 축소 → 마스터 팔레트 인덱싱 → 수동 정리** 파이프라인으로 생산한다. 한글 도트 폰트는 **갈무리11/9**.

주요 스프라이트 규격: **옥자(라이브 메인) 전신 약 128×288 — 프레임 애니/리깅이 아니라 "표정 스왑 스탠딩"(표정·포즈별 5~6장을 통째로 교체) + 코드 트윈(둥실 흔들·페이드)으로 구동**, 시온이(교감 가능한 펫) 48×48, 체키 카드 120×180(내부 아트 영역 104×140 고정), UI 아이콘 24×24, 재화 아이콘 16×16. 배경은 **나라카 지옥 배경(세계관 — 지옥 풍경) 270×480**. **게임기 셸 본체는 380×630**(≈1:1.65 클래식 핸드헬드 비율 — 내부 게임 화면 270×480 + 좌우 베젤 55 + 상단 브랜드 바 30 + 하단 버튼 덱 120). 폰 풀스크린을 채우지 않고 셸 본체를 세로 중앙 배치, 위아래 여백은 핑크 나라카 배경으로 채운다.

> 라이브 캐릭터를 프레임 시트로 그리거나 컷아웃 리깅하면 작업·난이도가 커진다. 미연시가 쓰는 스탠딩+표정차분 방식이 가장 쉽고, 본편 미연시와 에셋도 호환된다. 표정 5~6장은 같은 포즈·프레이밍으로 뽑아야 교체가 자연스럽다(→ PRD 리스크).

## 이유 / 트레이드오프

- **270×480**: 8비트 순수 레트로(예: 180×320)는 옥자님 얼굴·의상 디테일이 뭉개져 "감정 임팩트" 목표와 충돌. 반대로 고해상도는 도트 감성·작업 통일성을 잃음. 270×480은 얼굴 인식과 레트로 감성의 균형점이며 1080×1920에 정확히 ×4로 떨어진다.
- **마스터 팔레트 강제**: AI 도트화는 색이 제각각 튀어 한 게임처럼 안 보임. 인덱싱으로 통일감을 강제한다.
- **사진→도트 파이프라인**: 무예산·1주·일러스트레이터 미확보 상황에서, 카페가 이미 보유한 1년치 이벤트 사진을 활용하는 유일하게 현실적인 길. 고증·속도·일관성을 동시에 확보.

## 되돌리기 비용 (이 결정을 ADR로 남기는 이유)

해상도와 팔레트는 **한 번 정하면 전 에셋이 종속**된다. 나중에 바꾸면 모든 도트를 재작업해야 하므로, 초기에 못 박고 그 이유를 남긴다.
