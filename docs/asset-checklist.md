# 그래픽 에셋 체크리스트 (1주 데모)

> 출처: [ROADMAP.md](../ROADMAP.md) 아트 트랙(A1~A5) · [docs/adr/0001-dot-art-spec.md](./adr/0001-dot-art-spec.md) 규격 · [PRD.md](../PRD.md) §9.1 데모 이벤트.
> 파이프라인(1세트): **실물 사진 → AI 도트화 → nearest 축소 → 마스터 팔레트(~32색) 인덱싱 → 수동 정리**.
> 🛠 **생산 도구**: 이 체크리스트는 `tools/asset_manifest.json`으로 구조화되어 **도트 스튜디오 GUI**(`tools/dot_studio.py`)의 왼쪽 체크리스트 패널에 빈 슬롯으로 뜬다. 슬롯을 클릭해 규격·저장 경로를 자동 세팅하고 하나씩 채운다(완료 = 파일 존재). 슬롯 증감은 매니페스트에서 편집.

## 작업 원칙 (언제 만드나)

- **코드는 플레이스홀더로 먼저 돈다.** 단색 도형/임시 도트로 교감 루프를 완성하고, 에셋은 점진 교체 → **T20(Day 5~6)에서 전면 교체**.
- **아트는 별도 트랙으로 매일 누적.** 코드 진행을 막지 않는다.
- **A1이 최우선.** "링크 열면 옥자가 살아 움직인다"가 데모 합격 1번 조건. 옥자 표정 스탠딩 5~6장을 코어와 함께 가장 먼저.
- **표정 5~6장은 같은 포즈·프레이밍**으로 뽑아야 교체가 자연스럽다(→ ADR 0001 리스크).

## 총 물량 요약

- **옥자 스탠딩**: 표정 5~6장 + 이벤트 의상 **5세트** (지뢰계·유치원·힙합·집사·크리스마스) · 의상은 모두 체키 카드용 정적 아트(라이브 옥자는 마녀룩 고정)
- **시온이**: idle/반응 + 이벤트 의상 **2세트** (쿠로미·루돌프)
- **체키 카드(양면, 런타임 합성 → ADR 0003)**: 표지 세트 **4점**(파치먼트 + 날개·나비 엠블럼 + 나라카 워드마크, 전부 공용) + 뒷면 사진 프레임 = 표준 1 + 테마 **5** (메탈하트·크레용·그래피티·은쟁반·눈리스) + **사진 배경 5장**(`bg_cheki_*`, 의상별 — 누끼 옥자를 받치는 불투명 레이어)
- **배경 1** + **음료 3~4** + **선물 3~4** + 셸/아이콘/워터마크
