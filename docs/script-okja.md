# 옥자 보이스 스크립트 (데모 초안)

> 나라쿠치 — 옥자 대사/티커/대화 팝업 초안. 톤: **시크 · 츤데레 · 장난기**.
> 설계는 [PRD.md](../PRD.md) §4.3, 용어는 [CONTEXT.md](../CONTEXT.md) 참조.
>
> - `{닉}` = 플레이어 닉네임.
> - 호감: `↑↑` 큰 / `↑` 중 / `·` 작은(손해 아님, 벌점 없음).
> - **단계1(존댓말)** = 손님·**단골** + "{닉}님" / **단계2(반말)** = **편해진 사이**·마음 연 사이 + 닉네임. **존댓말→반말 전환은 편해진 사이(누적 600)** 에서 일어난다 — **단골(200)은 아직 단계1(존댓말)**, 단골 등극 컷인으로 한 템포 쌓는다(여전히 존댓말).
> - *이건 개발용 플레이스홀더. 실제 운영 시 옥자 톤으로 더 다듬을 것.* 라이브 대사 데이터는 `data/*.json`(ticker·talk·gifts)이며 `tools/content_studio.py`로 편집한다(이 문서는 톤 레퍼런스).

|---|---|---|
| 입장 | "또 왔네요, {닉}님." / "한가하신가 봐요." | "왔어, {닉}? 흥." / "{닉}야~ 늦었어." |
| 음료 주문 | "버건디예요. 마시세요." | "자, 버건디. 마셔." / "단골 서비스야." |

| 선물·선호 | "……어떻게 아셨죠." 😳 | "이거 좋아하는 건데. 고마워." 😳 |

## 💬 대화 팝업 토막 (`대화` 버튼, 가끔)

### [단계1] 잡담 A
> 옥자: "또 오셨네요. 일은 안 하세요?"
- 〔쉬러 왔어요〕 → "팔자 좋으시네요." 🙂 `↑`
- 〔마녀님 보러요〕 → "후, 입만 살아서는." 😏 `↑↑`
- 〔그냥요〕 → "성의 없으시긴." `·`

### [단계1] 잡담 B (시온이)
> 옥자: "시온이 예쁘죠. ……제 새끼라."
- 〔마녀님 닮았어요〕 → "……누가 누굴." 😳 `↑↑`
- 〔통통해요〕 → "먹는 게 일이라." 🙂 `↑`

### [💛 등극 컷인] 단골 등극 (손님 → 단골, **누적 200 도달 후 다음 입장**, 존댓말 유지)
> 옥자: "……{닉}님, 또 오셨네요."
> 옥자: 🙂 "이렇게 자주 와 주시는 분은 흔치 않은데 말이죠."
> 옥자: 😳 "뭐…… 단골 손님은 좀 챙겨 드릴게요."
>
> ▶ *단골 등극* (아직 존댓말 — 말투는 그대로, 살갑기만 +)

### [💜 전환 컷인] 존댓말 해제 (단계1 → 단계2, **편해진 사이 / 누적 600 도달 후 다음 입장**)
> 옥자: "{닉}님. ……아니."
> 옥자: 😳 "이제 그냥 **{닉}**이라고 부를게. 매일 오는데 님은 무슨."
> 옥자: "……착각 마. 부르기 편해서야."
>
> ▶ *반말 해금*

> 📝 두 컷인 모두 **다음 입장에서 1회** 발화하며, 대사·해금 줄·배지는 `data/ticker.json` 의 `okja_cutin[regular|comfy]` 에서 관리(콘텐츠 스튜디오 "옥자 티커" 탭 편집). 마음 연 사이(close, 2000)는 전용 연출 없이 평소 입장 인사(후속에 컷인 추가 가능).

### [단계2] 잡담 C
> 옥자: "왔어? ……오늘은 좀 보고 싶었는데."
- 〔나도〕 → "흥, 솔직하긴." 😳 `↑↑`
- 〔방금 뭐랬어?〕 → "못 들었으면 말고." 😤 `↑`
- 〔농담이지?〕 → "……됐어, 앉아." `·`

### [단계2] 잡담 D (선물 떡밥)
> 옥자: "요즘 시온이가 너만 따라. 뭐 먹였어?"
- 〔비밀이야〕 → "쳇, 치사해." 😏 `↑↑`
- 〔간식 줬어〕 → "고마워. 걔 잘 챙겨줘서." 😊 `↑↑`
