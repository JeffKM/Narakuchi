# 디바이스 세션 체크리스트 (T23 ② 실기기 · ③ 벽시계)

> 헤드리스로 못 잡는 항목만. 코드 검증(날짜 로직·영속 직렬화·공유 합성)은 `tools/run_tests.sh`로 완료(165통과). 대상: **실기기** — iOS Safari + 안드로이드 Chrome, `https://narakuchi.vercel.app`. (디버그 키는 release 빌드에 없음 — 실제 플레이로 확인.)

## ② 실기기 실측

### 1. 모바일 한글 IME (온보딩 닉네임)
- [ ] iOS Safari: 한글 조합 입력 정상(받침·이중모음)
- [ ] 마지막 글자 누락 없음(조합 중 확정 타이밍)
- [ ] 빈값으로 진행 → "손님" 폴백
- [ ] 안드 Chrome 동일 확인

### 2. 세이브 영속 (IndexedDB / 사파리)
- [ ] 교감 몇 번 → 탭 닫고 재진입 → 호감도·출석·체키 유지
- [ ] **iOS 홈 화면 추가(PWA) 모드**에서도 유지
- [ ] 사파리 강제종료 후 재실행 시 유지
- [ ] (참고) iOS Safari는 7일 미접속 시 스토리지 정리 정책 있음 — 데모엔 비치명

### 3. 공유 (Web Share API / JSBridge 폴백)
- [ ] 공유 버튼 → iOS 네이티브 공유 시트(파일 첨부) 뜸
- [ ] 미지원/취소 시 PNG 다운로드 폴백 동작
- [ ] 합성 이미지: 체키 + `@나라카` 워터마크 + QR 정상, 도트 깨짐 없음
- [ ] 안드 Chrome 공유/다운로드 확인

### 4. 터치 히트 영역
- [ ] 옥자 4버튼(체키·음료·대화·선물) 정확히 눌림
- [ ] 옥자 터치 / 시온이 탭 반응
- [ ] 셸 3버튼(SELECT/OK/CANCEL) 손가락 오발 없음
- [ ] 컬렉션북 탭·카드·공유 버튼 작은 타깃 확인

## ③ 밸런스 벽시계
- [ ] 하루치 한 세션(스태미나 6액션)을 스톱워치로 측정 → **1~3분** 체감 맞나
  - 코드 케이던스 측정값(참고): 단골 도달 알찬 3일·무난 5일 / 반말 알찬 7일·무난 14일

## (선택) 날짜 로직 실기기 가벼운 확인
- [ ] 어제 방문 후 오늘 재진입 → 출석 streak +1, 스태미나 풀 충전
- [ ] 24h+ 미접속 복귀 → 옥자 시무룩(1교감에 회복)
  - ⚠️ 관찰: `MOOD_PENALTY_HOURS=24` 정확 경계 — 매일 같은 시각 방문 시 트립. 조정 여부 미정.

## 발견 시 기록
- 버그/이상은 ROADMAP T23 ② 줄 또는 새 이슈로. 재배포는 `tools/deploy.sh`.
