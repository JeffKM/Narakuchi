# 시온 & 코코 & 규종: 캣닢 대탈환 작전 - 개발 로드맵

강아지 악당 콤비에게 빼앗긴 마법 캣닢 씨앗을 되찾는 8비트 레트로 아케이드 액션 게임의 MVP 개발 로드맵

## 개요

**캣닢 대탈환 작전**은 레트로 게임을 즐기는 캐주얼 게이머를 위한 8비트 횡스크롤 액션 게임으로 다음 기능을 제공합니다:

- **횡스크롤 액션 게임플레이**: 묵직한 탱크형 조작감의 시온 캐릭터로 좌우 이동, 점프, 슈퍼 엉덩이 찧기 스킬 구사
- **다양한 장애물과 미니보스**: 땅굴 함정, 쓰레기통, 바리케이드 등 3종 장애물과 선아 미니보스전
- **레트로 아케이드 경험**: 256x240 내부 해상도, 하트 시스템, 점수 시스템, 8비트 효과음

### 기술 스택

| 구분 | 기술 |
|------|------|
| 게임 엔진 | Phaser 3.90.0 |
| 웹 프레임워크 | Next.js 16 (App Router) |
| 언어 | TypeScript 5 |
| 스타일링 | TailwindCSS v4 |
| 오디오 | jsfxr (8비트 효과음) |
| 배포 | Vercel |
| 패키지 관리 | npm |

## 개발 워크플로우

1. **작업 계획**
   - 기존 코드베이스를 학습하고 현재 상태를 파악
   - 새로운 작업을 포함하도록 `ROADMAP.md` 업데이트
   - 우선순위 작업은 마지막 완료된 작업 다음에 삽입

2. **작업 생성**
   - 기존 코드베이스를 학습하고 현재 상태를 파악
   - `/tasks` 디렉토리에 새 작업 파일 생성
   - 명명 형식: `XXX-description.md` (예: `001-project-setup.md`)
   - 고수준 명세서, 관련 파일, 수락 기준, 구현 단계 포함
   - API/비즈니스 로직 작업 시 "## 테스트 체크리스트" 섹션 필수 포함
   - 초기 상태의 샘플로 `/tasks/000-sample.md` 참조

3. **작업 구현**
   - 작업 파일의 명세서를 따름
   - 기능과 기능성 구현
   - 각 구현 단계는 다음 사이클을 반복:
     1. 코드 구현
     2. 브라우저에서 수동 테스트 수행 (게임 동작 확인)
     3. 테스트 결과 검증 및 확인
     4. 실패 시 -> 재구현 후 재테스트 (통과할 때까지 반복)
     5. 통과 시 -> 다음 단계로 진행
   - 각 단계 후 작업 파일 내 단계 진행 상황 업데이트
   - 각 단계 완료 후 중단하고 추가 지시를 기다림

4. **로드맵 업데이트**
   - 로드맵에서 완료된 작업을 완료로 표시
